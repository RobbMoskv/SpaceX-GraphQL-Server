## How to use it?  
  
Clone the repository.
```
$ git clone https://github.com/RobbMoskv/SpaceX-GraphQL-Server.git
```

Download the dependencies.
```
$ npm install
```

Run the application locally on localhost:5000.
```
$ npm run
```
